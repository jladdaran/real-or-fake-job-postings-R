# Data Instructions

This project uses Kaggle's **Real or Fake Job Posting Prediction** dataset:

- **Kaggle Link:** https://www.kaggle.com/shivamb/real-or-fake-fake-jobposting-prediction
- **Rows:** 17,880  
- **Variables:** 18  
- **Dropped Columns:** job_id, department, salary_range  
- **Target Variable:** Fraudulent (0 = real, 1 = fake)

Place `fake_job_postings.csv` into `data/raw/`.

Tip: keep raw data out of Git; use `data/raw/` locally or Git LFS.
